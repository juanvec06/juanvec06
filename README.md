## 🚀 About Me

I'm a **Systems Engineering student (8th semester)** at Universidad del Cauca 🇨🇴, passionate about building backend systems that are not just functional — but **secure, scalable, and built to last**.

I spend most of my time thinking about clean architecture, robust APIs, and how to make services that hold up under real-world pressure. I've got solid hands-on experience with cloud services like **AWS**, and while backend is where my heart is, I can also navigate the frontend world with **Angular** and **React** when the project calls for it.

Lately, I'm genuinely excited about **AI agents** — I'm currently diving deep into **Strands Agents** and exploring how intelligent agents can solve meaningful problems. Speaking of which, I'm actively working on a project in collaboration with the **Gobernación del Cauca**, developing an **AI-powered emotional support assistant** for students at Universidad del Cauca — a project that pushes both my technical skills and my sense of purpose.

When I'm not coding, you'll find the rest of my work over on my repos — always building, always learning. 🛠️

- 🔐 Passionate about **secure & scalable backend development**
- ☁️ Building with **AWS** and cloud-native patterns
- 🤖 Currently exploring **AI agents with Strands Agents**
- 🌱 Always one side project away from learning something new
